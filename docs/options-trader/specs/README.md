# 设计 Spec 镜像 — 期权交易助手

本目录存放项目内 `docs/superpowers/specs/` 的**公开镜像**。

## 为什么要镜像

项目 repo 是 private（客户无权限看源码），但 spec 是值得对外展示的设计成果。在 docs-hub 留一份脱敏镜像可以：
- 给客户看设计质量和思考深度
- mkdocs 搜索能索引到
- 历史 spec 即使项目换架构也能追溯

## 脱敏要求

镜像前过全局 CLAUDE.md §18 脱敏红线：API key / 账户号 / 客户姓名 / 具体金额 / VM IP / 实盘参数具体值 都不能进 public。
